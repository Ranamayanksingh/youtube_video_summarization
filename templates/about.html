YT Summarizer

Turn any YouTube video into a clear, concise summary — in minutes, on your own machine.

What does it do?

Paste a YouTube link. The app does the rest — no manual work, no copying transcripts, no paying for a cloud API.

⬇️
Download

Fetches audio from any public YouTube video or the latest video from a channel.

🎙️
Transcribe

Converts speech to text using Whisper large-v3. Works with Hindi and English audio — always outputs English.

🧠
Summarize

Sends the transcript to a local LLM (llama3 via Ollama) and produces a structured summary.

💬
Ask questions

After summarizing, ask anything about the video. The LLM answers using the transcript as context.

Who can use it?

Anyone who has been granted access. Here's how:

  1. Web UI — Log in with the access token and use it from any browser on any device.
  2. Telegram bot — Message the bot directly from Telegram. Send a YouTube link, get a summary. No browser needed.
  3. Morning briefings — Subscribe to a YouTube channel. Every day at your chosen time, the latest video is summarized and sent to you on Telegram automatically.

Privacy first

Everything runs locally on a single Mac. No data leaves the machine — audio, transcripts, and summaries are stored on disk, processed by local models. There is no external AI API call.

✓ No cloud AI ✓ No data sent externally ✓ Runs on Apple Silicon ✓ Open to your own users only
