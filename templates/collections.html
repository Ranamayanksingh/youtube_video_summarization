How collections work

  1. Create a collection above and choose its goal type (Exam Prep, Quiz Practice, or Project Build).
  2. Send YouTube video links via the Telegram bot. After each summary, the bot asks you which collection to add it to.
  3. The system extracts formulas, questions, tricks, and concepts from each video transcript using AI.
  4. All extracted items are compiled into a growing knowledge/CollectionName.md file you can download and use anywhere.
  5. Use /quiz in the bot to get random questions, or /ask to query your entire knowledge base.
